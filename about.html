About the Happy Island Designer Tool

The Happy Island Designer tool was originally made by eugeneration when he saw the customization features in one of the AC directs. He later made it a project on GitHub and many people have used the fantastic tool. When I saw this tool I forked it as it was so good and now I've made some big changes to the project including adding a browser update banner, adding an android application to the project and even linking it to another project of mine, an island name generator tool. I hope to continue developing this project and helping out the community. I hope you guys love the tool as much as I do :D
